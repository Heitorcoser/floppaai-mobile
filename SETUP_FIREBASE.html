🔥 FloppaAI — Configurar Firebase

Siga este guia para conectar o app ao Firebase e receber tickets do celular em tempo real.

📋 Passo 1 — Criar o projeto Firebase

1. Acesse console.firebase.google.com

2. Clique em Adicionar projeto e dê um nome (ex: floppaai-tickets)

3. Desative o Google Analytics (opcional) e clique em Criar projeto

🗄 Passo 2 — Criar o Realtime Database

1. No menu lateral, clique em Build → Realtime Database

2. Clique em Create Database

3. Escolha a região United States (us-central1) e clique em Next

4. Selecione Start in test mode e clique em Enable

5. Copie a URL que aparece — ela será parecida com:

https://floppaai-tickets-default-rtdb.firebaseio.com

🔓 Passo 3 — Configurar as Regras de acesso

1. Na aba Rules do Realtime Database, substitua o conteúdo por:

{
  "rules": {
    "tickets": {
      ".read": false,
      ".write": true
    },
    "bugs": {
      ".read": false,
      ".write": true
    },
    "announcements": {
      ".read": true,
      ".write": false
    },
    "spam": {
      ".read": false,
      ".write": false
    }
  }
}

2. Clique em Publish

⚠️ Essas regras permitem que qualquer pessoa envie tickets, mas ninguém consegue ler os tickets — só o servidor (Python) com acesso direto consegue ler.

🖥 Passo 4 — Configurar o Dev Mode (PC)

1. Abra o FloppaAI Modo Desenvolvedor (dev_mode.py)

2. Vá em ⚙️ Configurações

3. Cole a URL do Firebase no campo Firebase URL

4. Clique em 💾 Salvar & Testar — deve aparecer "✅ Conectado!"

📱 Passo 5 — Configurar o Celular

1. Abra o mobile/index.html no navegador do celular

2. Na tela de setup que vai aparecer, cole a mesma URL do Firebase

3. Clique em Salvar e Continuar

4. Pronto! Qualquer ticket enviado pelo celular aparecerá no Dev Mode em segundos.

💡 Dica: Você pode hospedar o mobile/index.html gratuitamente no GitHub Pages ou Netlify para acessar de qualquer dispositivo sem copiar o arquivo.

✅ Fluxo completo:
Usuário no celular envia ticket → Firebase → Dev Mode no PC recebe em tempo real → Você responde → Usuário vê resposta na aba "Meus Tickets"
⚠️ O plano Spark (gratuito) do Firebase permite até 1 GB de armazenamento e 10 GB/mês de transferência. Para um app pequeno como o FloppaAI, isso é mais do que suficiente para sempre.
